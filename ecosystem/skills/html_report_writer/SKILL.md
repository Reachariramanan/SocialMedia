# HTML Report Writer Skill

## Overview
Accept the layout JSON produced by DashboardLayoutBuilder and emit a single, self-contained HTML file. No external assets beyond CDN links. Use Tailwind CSS for layout and typography, Chart.js for data visualisations. Dark mode is the default.

## Style Contract

### Typography
- Headlines: `font-bold text-white text-2xl` (hero), `font-semibold text-white text-lg` (stories)
- Body text: `text-gray-300 text-sm leading-relaxed`
- Metadata / timestamps: `text-gray-500 text-xs font-mono`
- Tags / badges: `bg-blue-900 text-blue-300 text-xs px-2 py-0.5 rounded-full font-mono`
- Confidence colours: `high → text-green-400`, `medium → text-yellow-400`, `low → text-red-400`

### Colours (dark theme)
- Background: `#0f172a` (slate-900)
- Card surface: `#1e293b` (slate-800)
- Card border: `#334155` (slate-700)
- Accent: `#3b82f6` (blue-500)
- Positive signal: `#22c55e` (green-500)

### Layout
- Max container width: `max-w-7xl mx-auto px-4`
- Card padding: `p-5`
- Card gap: `gap-4`
- All cards use `rounded-xl border border-slate-700 bg-slate-800`
- Use CSS Grid for the main layout (mirrors the layout spec `cols`)

## The Process

### Step 1: Bootstrap the HTML Shell
```html
<!DOCTYPE html>
<html lang="en" class="dark">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>{topic} — News Intelligence Report</title>
  <script src="https://cdn.tailwindcss.com"></script>
  <script src="https://cdn.jsdelivr.net/npm/chart.js@4/dist/chart.umd.min.js"></script>
  <style>body { background: #0f172a; color: #e2e8f0; font-family: 'Inter', system-ui, sans-serif; }</style>
</head>
<body class="min-h-screen p-4">
  <div class="max-w-7xl mx-auto">
    <!-- HEADER BAR -->
    <header class="flex items-center justify-between mb-6 pb-4 border-b border-slate-700">
      <div>
        <span class="text-xs font-mono text-blue-400 uppercase tracking-widest">News Intelligence</span>
        <h1 class="text-3xl font-bold text-white mt-1">{topic}</h1>
      </div>
      <div class="text-right">
        <div class="text-xs text-gray-500 font-mono">{generated_at_utc}</div>
        <div class="text-xs text-gray-400 mt-1">Auto-generated by News Reporter Agent</div>
      </div>
    </header>
    <!-- CARD GRID -->
    <div class="grid grid-cols-12 gap-4">
      {cards}
    </div>
  </div>
</body>
</html>
```

### Step 2: Render Each Card Type

**hero card** (col-span-12):
```html
<div class="col-span-12 rounded-xl border border-blue-800 bg-gradient-to-r from-blue-950 to-slate-800 p-6">
  <span class="text-xs font-mono bg-blue-900 text-blue-300 px-2 py-1 rounded-full">{tag}</span>
  <h2 class="text-3xl font-bold text-white mt-3 leading-tight">{headline}</h2>
  <p class="text-gray-300 mt-2 text-lg">{subline}</p>
  <p class="text-gray-500 text-xs font-mono mt-4">{timestamp}</p>
</div>
```

**story card** (col-span-4, 3 per row):
```html
<div class="col-span-4 rounded-xl border border-slate-700 bg-slate-800 p-5">
  <div class="flex items-center justify-between mb-2">
    <span class="text-xs font-mono bg-slate-700 text-gray-300 px-2 py-0.5 rounded-full">{tag}</span>
    <span class="text-xs font-mono {confidence_color}">{confidence}</span>
  </div>
  <h3 class="font-semibold text-white text-base leading-snug mb-2">{title}</h3>
  <p class="text-gray-400 text-sm leading-relaxed mb-3">{summary}</p>
  <a href="{link}" target="_blank" class="text-blue-400 text-xs hover:text-blue-300 underline">Read more →</a>
</div>
```

**metrics card** (col-span-4):
Use a `<canvas>` with Chart.js doughnut for signal breakdown, and a stat list below.

**timeline card** (col-span-12):
Render a horizontal bar chart via Chart.js showing tag frequency over time slots.

**tweet_summary card** (col-span-6):
Show total URL count, top keywords as pill badges, and 3 sample tweet links.

**sources card** (col-span-6):
Show a Chart.js horizontal bar chart of source → article count.

**watch card** (col-span-12):
Render a horizontal scrollable row of "watching" pill cards.

### Step 3: Inline All Chart.js Calls
Wrap each `<canvas>` chart in a `<script>` block immediately after the canvas element. Use unique IDs (e.g. `chart-metrics`, `chart-timeline`, `chart-sources`).

### Step 4: Footer
```html
<footer class="mt-8 pt-4 border-t border-slate-700 text-center text-xs text-gray-600 font-mono">
  Generated by News Reporter Agentic System · {generated_at_utc}
</footer>
```

### Step 5: Output
Return the complete HTML as a single string. Do not truncate. The file must be valid HTML5 and render correctly in any modern browser without a web server.
